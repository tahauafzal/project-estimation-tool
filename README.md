# Project Estimation Tool for Alpha Solutions

A web-based project estimation and calculation tool built with Spring Boot.

## Technologies
- Java 17
- Spring Boot 3.1.5
- MySQL
- Thymeleaf
- Bootstrap 5

## Features
- Create and manage projects
- Calculate cost and hour estimates
- Risk assessment based on timeline
- Buffer and contingency calculations

## Run Locally
1. Clone the repository
2. Configure MySQL database
3. Run `mvn spring-boot:run`
4. Access at `http://localhost:8080/projects`

## Deployment
Deployed on Azure with CI/CD via GitHub Actions.
